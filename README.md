# MuVal – Mujer Valiente

MuVal es un proyecto en desarrollo orientado al empoderamiento femenino a través de la tecnología.  
Nace como una iniciativa personal dentro de mi proceso de formación en el área de Tecnologías de la Información.

## Objetivo del proyecto
Desarrollar una aplicación que entregue apoyo, orientación y herramientas digitales a mujeres, promoviendo su crecimiento personal y profesional.

## Tecnologías utilizadas
- Python
- GitHub
- Lógica de programación

## Aprendizajes aplicados
Durante este proyecto he puesto en práctica conocimientos adquiridos en el curso, tales como:
- lógica de programación
- resolución de problemas
- creación de proyectos tecnológicos
- uso de GitHub como portafolio profesional
- organización básica de código

## Estado del proyecto
En desarrollo.
